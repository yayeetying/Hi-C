Container Stuff

Containers make use of the container class that is defined in Bootstrap.

Bootstrap containers make use of various breakpoints
That take effect when the size of the window reaches
A certain threshold.
Go ahead and resize the browser window
And notice how the text snaps to the left
At different widths of the window.
This happens when the container reaches its breakpoint, the width of the container becomes fixed rather than 100% of the browser window, causing it to snap to the left of the window. container-fluid will always have a width of 100% and container has various breakpoints rather than just one.

Grids utilize the classes row and col within a container.

This Grid has 2 rows, with varying columns in each row. There are a total of 12 different fixed column sizes (percentages of the page width). If you resize the page, the column with 4 will move to the next line, while the others stay on the same line. This is because that column makes use of variable width columns.
